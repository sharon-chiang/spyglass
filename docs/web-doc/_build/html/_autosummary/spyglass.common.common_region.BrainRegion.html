classmethod fetch_add(region_name, subregion_name=None, subsubregion_name=None)[source]#

Return the region ID for the given names, and if no match exists, first add it to the BrainRegion table.

The combination of (region_name, subregion_name, subsubregion_name) is effectively unique, then.

Parameters

  • region_name (str) – The name of the brain region.

  • subregion_name (str, optional) – The name of the subregion within the brain region.

  • subsubregion_name (str, optional) – The name of the subregion within the subregion.

Returns

region_id – The index of the region in the BrainRegion table.

Return type

int
